🌌 Project Nirak — Digital Engineering & Creative Infrastructure

Project Nirak ist eine unabhängige Entwicklungs- und Technologieinitiative von Nikolaus Flamann, mit Fokus auf nachhaltige Web-Infrastruktur, maßgeschneiderte Softwarelösungen und kreative Systemarchitektur.
Der Kern von Nirak ist die Verbindung von Präzision, Automatisierung und Ästhetik — vom Server-Kernel bis zur Benutzeroberfläche.

🧩 Technologie-Stack

- Pimcore Ecosystem: eigene Bundles wie SaturnBundle, SystemBundle, QuillSoftbreakBundle
- Backend-Engineering: PHP 8.3/8.4, Symfony, Laravel Hub-System, Doctrine, Redis
- Frontend-Development: Twig, Vue, Tailwind, Vite, Quill, modern UX-Patterns
- DevOps & Infra: Ubuntu LTS, Apache MPM Event, PHP-FPM Pools, RedisInsight, Matomo, Cloudflare, Hetzner Servers
- Automation & Compliance: Custom Shell Tools, DSGVO-konforme Analytics, NIS-2 Security, Open-Source Licensing (Nirak Custom License v1.0)

⚙️ Projekte & Module

- 🪐 SaturnBundle — Zentrales Pimcore-Bundle für Views, SEO, Dispatching und Twig-Helper
- 🖋 QuillSoftbreakBundle — Smarter Softbreak-Support für Pimcore Quill Editor
- 🔧 SystemBundle — Basisschicht für Configuration Services, Project Overlays & CLI-Tools
- 🌙 Hub System (Laravel) — zentrale OAuth2-Plattform für Benutzer- und Projektverwaltung

🧠 Philosophie

„Software darf nicht nur funktionieren – sie soll schön funktionieren.“
Project Nirak steht für kompromisslose Qualität, Transparenz und modulare Architektur.
Jede Komponente ist darauf ausgelegt, langfristig wartbar, auditierbar und erweiterbar zu bleiben.
